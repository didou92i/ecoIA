# Avis de tiers

## Runtime

L’extension distribuée n’embarque aucune dépendance JavaScript tierce. Le code source d’AI Wattch a
été examiné uniquement comme référence d’approche : aucun code source AI Wattch n’a été incorporé.

## Outils de développement

Ces outils ne sont pas inclus comme bibliothèques dans l’extension finale :

- Biome — licence MIT ou Apache-2.0 ;
- esbuild — licence MIT ;
- Playwright — licence Apache-2.0 ;
- TypeScript — licence Apache-2.0 ;
- Vitest — licence MIT ;
- jsdom — licence MIT ;
- types Chrome — licence MIT.

Le fichier `package-lock.json` fixe les versions exactes et contient les métadonnées des paquets.
Les textes de licence complets restent disponibles dans leurs distributions respectives.
